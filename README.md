# X-Intelligence

AML & Compliance screening portal built for clients and internal administrators of Xirni Limited.

## Project Structure

- `branding/`: Contains brand logo assets.
- `html/`: Web app containing:
  - Client Portal: Name screening, bulk screening, ongoing monitoring, case management, and reports.
  - Admin Portal: Masters, subscription management, package configuration, and user controls.
  - Core design system elements, utilities, page shell, and styling templates.

## Local Development & Deployment

The `html/` directory contains a static implementation configured with a `vercel.json` routing configuration to automatically point to `/login.html` and resolve clean URLs.
